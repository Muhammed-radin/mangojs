# ela.js 

 ela.js is a canvas 2d graphic libary for javascript.It is mainly used to make games.

##  table of contents

1. [import](README.md#-import)

## import

- import "ela.js" file
  
  in html: *add in head tag*
  ```html 
  <head>
    {... your codes ...}
    
    <script src="ela.js" type="text/javascript"></script>
  </head>
  ```

  in javascript: 
  ```js
    import ela from "/ela.js"
  ```
