Start Demo Project for Basics

the Mango is returning new mango object, if you also created mango object, add the canvas element to mango object.
example:-

          

 // creating new mango application
 var canvas = Mango()
 
 // setting canvas element to mango object
 var cnv = canvas.setCanvas(document.getElementById('c'));
 var c = cnv.elem;
 var ctx = cnv.ctx;
 var app = canvas.app;

use to easy way the entity and entity groups, entity and entity groups is class

          

 var entity = canvas.entity;
 var group = canvas.entityGroup;

creating a rectangle

          
            
 var rectangle = new entity({
   type: "rect",
   x:0,
   y: 0,
   width: 50,
   height: 50
 })

Render canvas Entities


canvas rendering with render() but render function is only one time rendering. so use repeatRender(), repeatRender is canvas clear and rendering, so you can make animation and use app.update 

          

 // one time render
 canvas.render()
 
 // clear canvas 
 canvas.clear()
 
 // loop render
 canvas.repeatRender()
 
 // if canvas is update 
 app.update = function (){
  // your codes
 }

# Application

application support and helping the canvas, this is using named canvas.app,canvas.app helping for you easily use to mango.js objects

application api

  • count(from, to, callback, howMany?, speed?) function
  • css(string) function
  • Vec2(x, y) class
  • randomColor(light?) function
  • randomNumberOnly(from, start) function
  • update property
  • html.input(type, x, y) function

# Entity

entityis sum to html element,see the api below

  • type String contents: rect, circle, clearRect, image, roundRect, path
  • name String
  • x Number
  • y Number
  • width Number
  • height Number
  • fill Color
  • stroke Color
  • render Boolean
  • scale Vec2
  • translate Vec2
  • rotate Angle
  • skew Vec2 (bugs found)
  • on Function
  • off Function
  • imageURl String
  • path String
  • strokeWidth Number
  • bx Number
  • by Number
  • dx Number
  • dy Number
  • id Number (auto generate)
  • radius Number
  • arcLevel Number
  • script String
  • afterScript Boolean
  • afterClip Boolean (bugs found)
  • shadow Number
  • shadowColor Color
  • shadowX Number
  • shadowY Number
  • text String
  • textMax Number
  • font String
  • fontSize Number
  • dWidth Number
  • dHeight Number
  • destroy Function
  • number Number (don't change this number)
  • physics Object (while crashing)
  • tags Array
  • imageSizeAuto Boolean
